## Language bindings
* C: There could not be any c bindings(see [Requirements](#requirements-to-import-puralingua))
* C++ [Supported Officially](https://github.com/PuraLingua/pura_lingua_cpp)


# Requirements to import Puralingua
* The language should be able to call C functions
* The language should have something like [RAII](https://en.wikipedia.org/wiki/Resource_acquisition_is_initialization) (e.g. C++, Java, C#, Rust)
